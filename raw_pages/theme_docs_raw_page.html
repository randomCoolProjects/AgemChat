Documentação sobre temas

Os demas devem seguir uma estrutura, sendo:
  • theme_dir
    • theme.json
    • icon.png
    • file(s)_to_replace.css

theme_dir: Diretório contendo os arquivos do tema. (O nome não importa)

theme_dir/theme.json: Arquivos mais importante. JSON especificando os atributos do tema.

theme_dir/icon.png: Ícone do tema.

theme_dir/file(s)_to_replace.css: Pode ser mais de um arquivo CSS. Estes arquivos irão substituir os estilos CSS padrão do AgemChat.

Este tema é um exemplo simples, que faz a animação "carregando" girar mais devagar.

Criando um tema

Para criar um tema você deve:
  • Conhecer os estilos CSS do AgemChat, que é open source. Consulte o Github.
  • Baixar o tema exemplo (citado acima) e veja como funciona.
  • Ter em mente a estrutura padrão de um tema (muito simples).

Enviando um tema

Ao "enviar" ume tema, tenha em mente que o AgemChat não o hospeda, apenas salva a URL do diretório de seu tema.
Na página "enviar tema", você verá dois campos:
No primeiro, digite o nome do seu tema, no segundo, a URL de seu diretório (Por exemplo, http://domain/theme-dir/). Após isso, clique em "Enviar tema".

Considerações finais

Note que:
  • O arquivo inteiro é substituído:
    • Se você criou um arquivo que irá atuar no lugar de "main.css", "main.css" todo será substituído, mesmo se você só mudou um detalhe.
  • Temas inválidos não serão aceitos:
    • Certifique-se que todos os padrões são seguidos, caso contrário, seu tema não será aceito ou será removido posteriormente.
